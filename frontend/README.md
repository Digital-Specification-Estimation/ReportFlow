# ReportFlow Frontend

## Project Overview

ReportFlow is a comprehensive report/appraisal management system with a React frontend and NestJS backend.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- TipTap (Rich text editor)
- React Router
- Zustand (State management)

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Development

The frontend runs on `http://localhost:8080` by default and connects to the NestJS backend API.
